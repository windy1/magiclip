# magiclip

## Installing

On Linux:
```bash
$ sudo apt install xorg-dev libxcb-shape0-dev libxcb-xfixes0-dev clang
```

```bash
$ git clone https://github.com/windy1/magiclip
$ cd magiclip
$ cargo install --path magiclip
$ sudo ./magiclipd/install.sh
$ sudo ./magiclipd/start.sh
```

## Running

```bash
$ magiclip
```
